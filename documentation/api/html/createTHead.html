HTML API

createTHead

Add a table header element to a specified table element.
Throws an exception when the application cannot find a match for the table element or the innerHTML is different. See HTML live edit for further information.

tilepieces.core.htmlMatch.createTHead(tableEl);

Parameters

tableEl
A table element

Return Value

None.
