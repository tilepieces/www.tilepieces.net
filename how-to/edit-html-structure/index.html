Change document structure

In this tutorial we know the component panel, which allows us to insert tags and data structures of components.
We pay particular attention to the insertion mode value, this is what decides the position where a new tag will go (in relation to the selected element).

The html inspector panel, in addition to allowing us to see the structure of our document, allows us to make further changes, including moving, deleting a node, editing a text node and adding attributes to the nodes
