How to create and reuse a component

The tutorial above illustrates how to create and reuse a component.

Components can be saved and imported. The local components and global components sections within the package manager panel help us to reuse it.
A component must be imported as local to be used in a project.

All front end projects (intended for use in a browser) can be managed by tilepieces.

In this git repository, there is a component collection project.
You can download the zip to import them from here.
