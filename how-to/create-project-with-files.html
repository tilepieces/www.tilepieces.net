How to create a project and add files

As a first step, let's start creating a project.

To do this, go to the package manager panel and click the create button in the first tab.

Once done, a new project will be created with an index.html file inside it.

To add files within the project, use the project explorer panel, as shown in the video.

With the first we create a new project, while with the second we go to insert an html file called "a-test".
We are ready to start!
