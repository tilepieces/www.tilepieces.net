change text and media tags

Tilepieces offers all the tools to edit preset text and edit IMG and VIDEO tag links directly from its basic interface.

Through the text editor icon button it will be possible to edit the text of your files.You can paste some text, but be aware that the browser will add styles to it in addition to copying the text. Browsers allow a "paste as text only" function to avoid this.

To edit IMG and VIDEO tags, you can copy / paste or drag and drop your files directly onto items when you select them with the element selection icon button  . Then a dialog will appear asking you how to call the file that will be saved in your project.
In the video above you will see an example of drag and drop with an IMG tag.
